# 项目结构

+ 客户端文件夹 
  + custom 存储用户定制相关的内容
  + library 存放程序运行时的静态依赖库
  + projects 默认的ini项目存放位置
  + resources 默认的ini媒体资源文件存放的位置
  + config.json 存放客户端配置文件
 
+ 源代码文件
  + electron文件夹存放electron的后端代码
  + env vite环境
  + src 前端源文件夹  
     - components 可复用组件
     - pages 应用页面文件
     - states UI的全局状态
     - stores 数据仓库
